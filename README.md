## 🎯 两大核心功能

### 1️⃣ 音视频转录

支持从音视频文件直接提取语音并生成字幕：

- **🎬 多格式支持** - MP3、MP4、WAV、M4A、WEBM 等常见格式
- **🤖 本地转录** - 使用 AI 模型在浏览器中本地转录，保护隐私
- **🌍 多语言支持** - 支持英语及 25 种欧洲语言
- **⚡ GPU 加速** - 支持 WebGPU 硬件加速，转录速度更快
- **📊 实时进度** - 显示解码、分片、转录各阶段进度

#### 🧠 转录模型：NVIDIA Parakeet TDT 0.6B

采用 NVIDIA 开源的 Parakeet 模型（600M 参数）：

- ✨ **智能识别** - 自动标点、大小写，准确识别数字和歌词
- 🎯 **精确时间戳** - 词级时间戳，字幕同步精准
- ⏱️ **无限时长** - 自动分片处理，支持任意长度音视频
- 📜 **开源许可** - CC BY 4.0，商业/非商业均可免费使用

**可选模型版本**：

| 模型 | 语言支持 | 推荐场景 |
|------|----------|----------|
| **V2** | 英语 | 英语内容，准确率稍高 |
| **V3** | 25 种欧洲语言（含英语，自动检测） | 多语言或混合内容 |

> 💡 **首次使用**需下载约 2.3 GB 模型文件，下载后永久缓存

### 2️⃣ 字幕翻译

智能翻译 SRT 字幕文件：

- **🧠 AI 驱动** - 基于大语言模型，翻译质量高
- **🌍 多语言互译** - 中文、英文、日语、韩语等
- **📚 术语管理** - 自定义词典，专业术语翻译更准确
- **📖 历史记录** - 自动保存翻译结果，随时重新导出
- **⚡ 批量处理** - 同时处理多个字幕文件

## 🚀 快速开始

### 方式一：音视频转字幕

1. 点击「上传音视频」按钮
2. 选择要转录的音视频文件
3. 点击「下载模型」（首次使用，约 2.3 GB，只需下载一次）
4. 等待转录完成，自动生成 SRT 字幕文件
5. 可选择继续翻译字幕

### 方式二：翻译已有字幕

1. 点击「上传字幕」按钮
2. 选择 SRT 字幕文件
3. 配置翻译 API（支持 DeepSeek、通义千问、智谱、豆包等）
4. 选择源语言和目标语言
5. 点击「开始翻译」，完成后下载结果

## ❓ 常见问题

**Q: 转录需要多久？**
A: 取决于音频长度和设备性能。一般 10 分钟音频约需 2-5 分钟（GPU 加速）。

**Q: 支持哪些语言的转录？**
A: V2 模型支持英语。V3 模型支持 25 种欧洲语言，包括：英语、法语、德语、西班牙语、意大利语、葡萄牙语、俄语、乌克兰语、荷兰语、波兰语等（自动检测语言）。

**Q: 必须配置 API 吗？**
A: 转录功能不需要 API。翻译功能需要配置兼容 OpenAI 格式的 API 服务。

**Q: 模型需要下载多久？**
A: 首次使用需下载约 2.3 GB，根据网速一般需要 5-15 分钟。下载后永久缓存。
